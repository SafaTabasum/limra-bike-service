## 📝 Description

LIMRA is a professional two-wheeler service website showcasing 40+ years of experience in bike repair and maintenance. The website features a modern design with animations, Google Maps integration for branch locations, and comprehensive service information.

## ✨ Features

- **Responsive Design** - Works perfectly on all devices (desktop, tablet, mobile)
- **Interactive Hero Section** - Auto-sliding bike images (BMW, Royal Enfield, Jawa, Activa)
- **Service Cards** - Animated cards displaying all bike services
- **Location Maps** - Google Maps integration for each branch
- **Opening Hours** - Clear timings with Friday special hours
- **Contact Section** - Click-to-call phone numbers
- **Smooth Animations** - Scroll-triggered animations throughout
- **Sticky Header** - Navigation header with scroll effect

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- JavaScript (ES6)
- Google Maps Embed API

## 🚀 How to Use

1. Clone the repository:
```bash

```

2. Navigate to project folder:
```bash

```

3. Open `index.html` in your browser or use Live Server.

## 💻 Local Development

To run this project locally with live reload:

```bash

python -m http.server 8000

# Right-click on index.html → Open with Live Server
```
